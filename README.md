# cumbria-wellness

A simple public index of wellness-related resources for Cumbria / the South Lakes — currently a one-page weekly **yoga** calendar for the Kendal / Levens area.

**Live site:** https://dwmkerr.github.io/cumbria-wellness/

## What this is

`index.html` is a single, self-contained page (no build, no dependencies): a weekly grid of regular local yoga classes. Click any class for its venue, address, Google Maps link, studio website and a short description. Green = confirmed against the studio's own timetable; amber = verify first.

It's a personal, community-minded index — times change, so always confirm with the studio.

## Publishing (CI/CD)

GitHub Pages serves `index.html` from the `main` branch root. **Pushing to `main` republishes automatically** — no build step. To update, replace `index.html` and push.
